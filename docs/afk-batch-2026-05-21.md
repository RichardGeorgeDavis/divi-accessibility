# AFK Batch 2026-05-21

## 2.1.10 Dropdown Reverse-Tab Focus Cleanup

- Upstream report: <https://github.com/campuspress/divi-accessibility/issues/123>
- Reported behavior: desktop dropdown submenus close correctly when tabbing
  forward through top-level menu items, but remain open when navigating backward
  with Shift+Tab.
- Fix: when focus enters a menu link or dropdown helper button, close all
  dropdown branches outside the newly focused menu branch; when focus leaves a
  menu branch entirely, close that branch and its descendants.
- Release tooling fix: updated `scripts/prerelease.js` so release metadata
  checks support multi-digit patch versions such as `2.1.10`.
