# Support

## Questions

Use GitHub Discussions Q&A for setup, usage, and compatibility questions:

https://github.com/RichardGeorgeDavis/divi-accessibility/discussions/categories/q-a

If Discussions are unavailable, open a GitHub issue with the `question` context
and include enough detail to reproduce the behavior.

## Bug Reports

For bugs, include:

- WordPress version
- Divi version
- Divi Accessibility version
- active theme or child theme
- relevant active plugins
- browser and assistive technology, if applicable
- steps to reproduce
- expected behavior
- actual behavior

## Security

Do not report security vulnerabilities in public issues. See `SECURITY.md`.

## Project Status

This is a maintained community fork unless official ownership or WordPress.org
committer access is granted. Upstream review remains welcome through the
canonical release-candidate pull request.
